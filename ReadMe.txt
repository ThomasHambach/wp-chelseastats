== Description ==

Adds a sidebar widget showing latest Chelsea Stats and form, using twitter bootstrap for style/scripts.

Settings page allows you to set what stats to display and how many results accounted for in the form guide as well disabling twitter bootstrap if you are using it already.

More charts and data coming soon.

uses the api.thechels.co.uk and requires registering (it's free).

Compatible with WordPress 3.0+.

== Installation ==

1. Upload the .php file to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Access widgets in the admin and drag widget to widget/sidebar area.
4. visit admin page to choose display options
